# shigeifinder

**Tags:** <a href="/developers/tags/shigella" class="badge badge--primary badge--tag">shigella</a> <a href="/developers/tags/eiec" class="badge badge--primary badge--tag">eiec</a> <a href="/developers/tags/serotype" class="badge badge--primary badge--tag">serotype</a> <a href="/developers/tags/typing" class="badge badge--primary badge--tag">typing</a> <a href="/developers/tags/cluster-analysis" class="badge badge--primary badge--tag">cluster-analysis</a> <a href="/developers/tags/sample-scope" class="badge badge--primary badge--tag">sample-scope</a> 

Prever sorotipos de Shigella e EIEC a partir de montagens.

Este subworkflow utiliza o [ShigEiFinder](https://github.com/LanLab/ShigEiFinder) para prever
sorotipos de *Shigella* e *E. coli* Enteroinvasiva (EIEC) a partir de genomas montados.
Ele usa uma abordagem baseada em clusters para identificar marcadores específicos de sorotipo e classificar
isolados com base em seus perfis antigênicos.

## Entrada

```
assembly: Channel<Record>
```

| Campo | Descrição |
|-------|-----------|
| `meta` | Registro Groovy contendo informações da amostra |
| `assembly` | Contigs montados no formato FASTA |

## Saída

### Publicados

As emissões `sample_outputs` e `run_outputs` são agregados de arquivos de saída que serão publicados no fluxo de trabalho principal.

#### `sample_outputs`

| Saída | Descrição |
|-------|-----------|
| `tsv` | Resultados do ShigEiFinder no formato TSV |

#### `run_outputs`

| Saída | Descrição |
|-------|-----------|
| `csv` | Resultados agregados no formato CSV |
