# legsta

**Tags:** <a href="/developers/tags/legionella" class="badge badge--primary badge--tag">legionella</a> <a href="/developers/tags/pneumophila" class="badge badge--primary badge--tag">pneumophila</a> <a href="/developers/tags/sequence-typing" class="badge badge--primary badge--tag">sequence-typing</a> <a href="/developers/tags/st" class="badge badge--primary badge--tag">st</a> <a href="/developers/tags/epidemiology" class="badge badge--primary badge--tag">epidemiology</a> <a href="/developers/tags/sample-scope" class="badge badge--primary badge--tag">sample-scope</a> 

Tipagem baseada em sequência de Legionella pneumophila in silico.

Este subworkflow realiza a tipagem baseada em sequência de Legionella pneumophila
usando [legsta](https://github.com/tseemann/legsta), que identifica o
Tipo de Sequência (ST) com base no esquema de sete loci. A ferramenta analisa
perfis de alelos e fornece dados de tipagem epidemiológica para investigação
de surtos e estudos populacionais.

## Take

```
assembly: Channel<Record>
```

| Campo | Descrição |
|-------|-----------|
| `meta` | Registro Groovy contendo informações da amostra |
| `assembly` | Arquivos de montagem em formato FASTA para tipagem de sequência de L. pneumophila |

## Emit

### Publicados

As emissões `sample_outputs` e `run_outputs` são agregados de arquivos de saída que serão publicados no fluxo de trabalho de entrada.

#### `sample_outputs`

| Saída | Descrição |
|-------|-----------|
| `tsv` | Resultados de SBT de Legionella pneumophila delimitados por tabulação com números de alelos e tipo de sequência |

#### `run_outputs`

| Saída | Descrição |
|-------|-----------|
| `csv` | Resultados agregados em formato CSV |
