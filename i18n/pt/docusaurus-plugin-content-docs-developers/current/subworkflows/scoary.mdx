# scoary

**Tags:** <a href="/developers/tags/gwas" class="badge badge--primary badge--tag">gwas</a> <a href="/developers/tags/association" class="badge badge--primary badge--tag">association</a> <a href="/developers/tags/pan-genoma" class="badge badge--primary badge--tag">pan-genoma</a> <a href="/developers/tags/traits" class="badge badge--primary badge--tag">traits</a> <a href="/developers/tags/statistical" class="badge badge--primary badge--tag">statistical</a> <a href="/developers/tags/run-scope" class="badge badge--primary badge--tag">run-scope</a> 

Estudos de associação pan-genômica ampla.

Este subworkflow realiza estudos de associação genômica ampla (GWAS) em dados de pan-genoma
utilizando o [Scoary](https://github.com/AdmiralenOla/Scoary). A ferramenta identifica genes
associados a características binárias, como patogenicidade, especificidade de hospedeiro ou
resistência a antibióticos. Ele calcula associações estatísticas entre presença/ausência de genes
e características fenotípicas em múltiplos isolados bacterianos.

## Entrada

```
csv: Channel<Record>
```

| Campo | Descrição |
|-------|-----------|
| `meta` | Registro Groovy contendo informações da amostra |
| `csv` | Matriz de presença/ausência de genes da análise de pan-genoma em formato CSV |

```
traits: Path?
```

| Nome | Tipo | Descrição |
|------|------|-----------|
| `traits` | `Path?` | Arquivo de características contendo características fenotípicas binárias para cada isolado (opcional) |

## Saída

### Publicados

As emissões `sample_outputs` e `run_outputs` são agregados de arquivos de saída que serão publicados no fluxo de trabalho de entrada.

#### `sample_outputs`

Nenhuma saída no escopo de amostra.

#### `run_outputs`

Nenhuma saída no escopo de execução.
