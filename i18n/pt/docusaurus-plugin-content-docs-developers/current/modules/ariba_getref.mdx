# ariba_getref

**Tags:** <a href="/developers/tags/bacteria" class="badge badge--primary badge--tag">bacteria</a> <a href="/developers/tags/database" class="badge badge--primary badge--tag">database</a> <a href="/developers/tags/download" class="badge badge--primary badge--tag">download</a> <a href="/developers/tags/antimicrobial-resistance" class="badge badge--primary badge--tag">antimicrobial-resistance</a> <a href="/developers/tags/virulence" class="badge badge--primary badge--tag">virulence</a> <a href="/developers/tags/ariba" class="badge badge--primary badge--tag">ariba</a> <a href="/developers/tags/setup" class="badge badge--primary badge--tag">setup</a> <a href="/developers/tags/run-scope" class="badge badge--primary badge--tag">run-scope</a> 

Baixe e prepare bancos de dados de referência para análise com ARIBA.

Utiliza o [ARIBA](https://github.com/sanger-pathogens/ariba) para buscar bancos de dados de referência curados
(por exemplo, CARD, ResFinder, VFDB, PlasmidFinder) e prepará-los para detecção de genes baseada em montagem local.
O banco de dados é indexado e empacotado em um arquivo tarball para uso com `ariba run`.

:::note[Conexão com a Internet Necessária]
Este processo requer uma conexão ativa com a internet para baixar o banco de dados especificado.
:::

## Entradas

```
db_name: String
```

| Nome | Tipo | Descrição |
|------|------|-----------|
| `db_name` | `String` | Nome do banco de dados a ser baixado (por exemplo, 'card', 'resfinder', 'vfdb_core', 'plasmidfinder') |

## Saídas

```
record (
    db: Path,
    logs: Set<Path?>
)
```

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `db` | `Path` | Um arquivo tarball comprimido contendo o banco de dados ARIBA preparado |
| `logs` | `Set<Path?>` | Arquivos de log opcionais específicos do programa |
