# amrfinderplus_update

**Tags:** <a href="/developers/tags/bacteria" class="badge badge--primary badge--tag">bacteria</a> <a href="/developers/tags/database" class="badge badge--primary badge--tag">database</a> <a href="/developers/tags/antimicrobial-resistance" class="badge badge--primary badge--tag">antimicrobial-resistance</a> <a href="/developers/tags/update" class="badge badge--primary badge--tag">update</a> <a href="/developers/tags/download" class="badge badge--primary badge--tag">download</a> <a href="/developers/tags/ncbi" class="badge badge--primary badge--tag">ncbi</a> <a href="/developers/tags/sample-scope" class="badge badge--primary badge--tag">sample-scope</a> 

Baixe e indexe o banco de dados mais recente do AMRFinder+.

Obtém os bancos de dados mais recentes do [AMRFinder+](https://github.com/ncbi/amr) a partir do NCBI,
indexa-os e os empacota em um arquivo tarball.

:::note[Manutenção Interna]
Este processo é utilizado principalmente de forma interna pelo Bactopia para construir e atualizar os
conjuntos de dados integrados.
:::

## Saídas

```
record (
    db: Path,
    logs: Set<Path?>
)
```

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `db` | `Path` | Um tarball comprimido do banco de dados mais recente do AMRFinder+ |
| `logs` | `Set<Path?>` | Arquivos de log opcionais específicos do programa |
