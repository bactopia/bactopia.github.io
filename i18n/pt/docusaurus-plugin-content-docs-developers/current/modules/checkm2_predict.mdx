# checkm2_predict

**Tags:** <a href="/developers/tags/quality-control" class="badge badge--primary badge--tag">quality-control</a> <a href="/developers/tags/completeness" class="badge badge--primary badge--tag">completeness</a> <a href="/developers/tags/contamination" class="badge badge--primary badge--tag">contamination</a> <a href="/developers/tags/machine-learning" class="badge badge--primary badge--tag">machine-learning</a> <a href="/developers/tags/bacteria" class="badge badge--primary badge--tag">bacteria</a> <a href="/developers/tags/archaea" class="badge badge--primary badge--tag">archaea</a> <a href="/developers/tags/sample-scope" class="badge badge--primary badge--tag">sample-scope</a> 

Avalie a qualidade do genoma usando aprendizado de máquina.

Utiliza o [CheckM2](https://github.com/chklovski/CheckM2) para prever a completude e
a contaminação de montagens de genomas. Ao contrário do CheckM original, ele usa um modelo
de aprendizado de máquina de gradient boost para prever a qualidade sem depender de conjuntos
de marcadores específicos de linhagem, tornando-o mais preciso para genomas novos ou reduzidos.

:::note[Banco de Dados Necessário]
Requer que o banco de dados do CheckM2 (arquivo de banco de dados Diamond) esteja disponível.
:::

```

    meta: Record,

```

| Campo | Tipo | Descrição |
|-------|------|-----------|

| `logs` | `Set<Path?>` | Arquivos de log opcionais específicos do programa |
| `nf_logs` | `Set<Path>` | Arquivos de log específicos do Nextflow (ex.: .command.&#123;begin|err|log|out|run|sh|trace&#125;) |
| `versions` | `Set<Path>` | Um arquivo formatado em YAML com as versões dos programas |

## Parâmetros

### Parâmetros do CheckM2

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `--checkm2_lowmem` | boolean |  | Modo de baixa memória. Reduz o tamanho do bloco do DIAMOND para diminuir significativamente o uso de RAM, em troca de um tempo de execução maior |
| `--checkm2_general` | boolean |  | Força o uso do modelo geral de previsão de qualidade (gradient boost) |
| `--checkm2_specific` | boolean |  | Força o uso do modelo específico de previsão de qualidade (rede neural) |
| `--checkm2_allmodels` | boolean |  | Gera a previsão de qualidade de ambos os modelos para cada genoma |
| `--checkm2_genes` | boolean |  | Trata os arquivos de entrada como arquivos de proteínas. [Padrão: False] |
| `--checkm2_opts` | string |  | Opções adicionais para passar ao CheckM2 |
