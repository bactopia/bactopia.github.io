# rgi_heatmap

**Tags:** <a href="/developers/tags/resistance" class="badge badge--primary badge--tag">resistance</a> <a href="/developers/tags/antimicrobial-resistance" class="badge badge--primary badge--tag">antimicrobial-resistance</a> <a href="/developers/tags/card" class="badge badge--primary badge--tag">card</a> <a href="/developers/tags/rgi" class="badge badge--primary badge--tag">rgi</a> <a href="/developers/tags/heatmap" class="badge badge--primary badge--tag">heatmap</a> <a href="/developers/tags/visualization" class="badge badge--primary badge--tag">visualization</a> <a href="/developers/tags/run-scope" class="badge badge--primary badge--tag">run-scope</a> 

Crie mapas de calor de presença/ausência de genes de resistência.

Utiliza o [RGI](https://github.com/arpcard/rgi) (Resistance Gene Identifier) para gerar
mapas de calor que visualizam a presença ou ausência de genes de resistência antimicrobiana
em múltiplas amostras com base nos resultados JSON do RGI.

## Entradas

```
record (
    meta: Record,
    json: Set<Path>
)
```

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `meta` | `Record` | Registro Groovy contendo informações da amostra |
| `json` | `Set<Path>` | Lista de resultados do RGI em formato JSON |

## Saídas

```
record (
    meta: Record,
    heatmap: Set<Path?>,
    results: Set<Path>,
    logs: Set<Path?>,
    nf_logs: Set<Path>,
    versions: Set<Path>
)
```

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `meta` | `Record` | Registro com informações da amostra |
| `heatmap` | `Set<Path?>` | Arquivos de mapa de calor em vários formatos (CSV, EPS, PNG) |
| `results` | `Set<Path>` | Todos os arquivos de saída a serem publicados |
| `logs` | `Set<Path?>` | Arquivos de log específicos do programa (opcionais) |
| `nf_logs` | `Set<Path>` | Arquivos de log específicos do Nextflow (ex.: .command.&#123;begin|err|log|out|run|sh|trace&#125;) |
| `versions` | `Set<Path>` | Arquivo no formato YAML com as versões dos programas |
