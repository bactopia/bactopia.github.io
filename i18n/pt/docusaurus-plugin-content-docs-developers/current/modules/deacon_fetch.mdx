# deacon_fetch

**Tags:** <a href="/developers/tags/host" class="badge badge--primary badge--tag">host</a> <a href="/developers/tags/decontamination" class="badge badge--primary badge--tag">decontamination</a> <a href="/developers/tags/depletion" class="badge badge--primary badge--tag">depletion</a> <a href="/developers/tags/download" class="badge badge--primary badge--tag">download</a> <a href="/developers/tags/index" class="badge badge--primary badge--tag">index</a> <a href="/developers/tags/minimizer" class="badge badge--primary badge--tag">minimizer</a> <a href="/developers/tags/deacon" class="badge badge--primary badge--tag">deacon</a> <a href="/developers/tags/run-scope" class="badge badge--primary badge--tag">run-scope</a> 

Baixa um índice deacon pré-construído para filtragem de reads do hospedeiro.

Utiliza o [deacon](https://github.com/bede/deacon) para baixar um índice minimizer pré-construído
para filtragem de reads do hospedeiro a partir de dados de sequenciamento. O índice padrão é `panhuman-1`, que
cobre genomas de referência humanos para depleção de reads humanas.

:::note[Conexão com a Internet Necessária]
Este processo requer uma conexão ativa com a internet para baixar o índice pré-construído.
:::

## Saídas

```
record (
    db: Path,
    logs: Set<Path?>
)
```

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `db` | `Path` | O arquivo de índice minimizer deacon pré-construído |
| `logs` | `Set<Path?>` | Arquivos de log opcionais específicos do programa |

## Parâmetros

### Parâmetros do Deacon Fetch

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `--deacon_index_name` | string | `panhuman-1` | Nome do índice deacon pré-construído a ser baixado |
| `--download_deacon` | boolean | `false` | Baixa o índice deacon para o cache de datasets |
| `--use_deacon` | boolean | `false` | Usa o deacon para filtragem de reads do hospedeiro |
