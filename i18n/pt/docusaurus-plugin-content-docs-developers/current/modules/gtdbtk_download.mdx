# gtdbtk_download

**Tags:** <a href="/developers/tags/gtdb" class="badge badge--primary badge--tag">gtdb</a> <a href="/developers/tags/taxonomy" class="badge badge--primary badge--tag">taxonomy</a> <a href="/developers/tags/database" class="badge badge--primary badge--tag">database</a> <a href="/developers/tags/download" class="badge badge--primary badge--tag">download</a> <a href="/developers/tags/setup" class="badge badge--primary badge--tag">setup</a> <a href="/developers/tags/bacteria" class="badge badge--primary badge--tag">bacteria</a> <a href="/developers/tags/archaea" class="badge badge--primary badge--tag">archaea</a> <a href="/developers/tags/sample-scope" class="badge badge--primary badge--tag">sample-scope</a> 

Baixe e configure o banco de dados de referência do GTDB-Tk.

Utiliza o script oficial `download-db.sh` para buscar os arquivos mais recentes do Genome Taxonomy Database (GTDB)
necessários pelo [GTDB-Tk](https://github.com/Ecogenomics/GTDBTk). O processo descomprime automaticamente
os dados e verifica a instalação usando `gtdbtk check_install`.

:::note[Conexão com a Internet e Espaço em Disco Necessários]
Este processo requer uma conexão ativa com a internet e espaço em disco significativo (~60GB+ descomprimido)
para armazenar os arquivos do banco de dados.
:::

## Saídas

```
record (
    db: Path?,
    db_tarball: Path?,
    logs: Set<Path?>
)
```

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `db` | `Path?` | O diretório contendo os arquivos do banco de dados GTDB-Tk descomprimidos |
| `db_tarball` | `Path?` | Um tarball comprimido do banco de dados (se solicitado via parâmetros) |
| `logs` | `Set<Path?>` | Arquivos de log opcionais específicos do programa |

## Parâmetros

### Parâmetros de Configuração do Banco de Dados GTDB-Tk

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `--gtdb` | string |  | Tarball ou caminho de um banco de dados GTDB. Se um banco de dados não for encontrado, você deve usar '--download_gtdb' |
| `--download_gtdb` | boolean | `false` | Baixa o banco de dados GTDB mais recente, mesmo que já exista |
| `--gtdb_save_as_tarball` | boolean | `false` | Baixa o banco de dados GTDB mais recente e o salva em um único tarball |
