# bactopia-cleanup-coverage

Reduzir redundância na cobertura por base a partir da saída do genomeCoverageBed.

## Uso

```bash
bactopia-cleanup-coverage COVERAGE [OPTIONS]
```

## Argumentos

| Argumento | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `COVERAGE` | STRING | Sim | O arquivo de cobertura por base do genomeCoverageBed. |

## Opções

| Opção | Tipo | Padrão | Descrição |
|-------|------|--------|-----------|
| `--version, -V` | BOOL | `false` | Exibir a versão e sair. |
