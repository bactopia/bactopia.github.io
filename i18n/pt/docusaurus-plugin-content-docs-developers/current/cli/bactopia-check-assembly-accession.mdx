# bactopia-check-assembly-accession

Verificar se o acesso de montagem do NCBI é o mais recente e ainda está disponível.

## Uso

```bash
bactopia-check-assembly-accession REFERENCE [OPTIONS]
```

## Argumentos

| Argumento | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `REFERENCE` | STRING | Sim | O acesso de montagem a ser verificado. |

## Opções

| Opção | Tipo | Padrão | Descrição |
|-------|------|--------|-----------|
| `--version, -V` | BOOL | `false` | Exibe a versão e sai. |
