Bactopia Tool: Plasmidfinder.

Identificação de plasmídeos a partir de montagens
O módulo `plasmidfinder` identifica plasmídeos em isolados bacterianos total ou parcialmente sequenciados.

## Uso

Bactopia CLI:

```bash
bactopia --wf plasmidfinder \
  --bactopia /path/to/your/bactopia/results
```

Nextflow:

```bash
nextflow run bactopia/bactopia/workflows/bactopia-tools/plasmidfinder/main.nf \
  --bactopia /path/to/your/bactopia/results
```
