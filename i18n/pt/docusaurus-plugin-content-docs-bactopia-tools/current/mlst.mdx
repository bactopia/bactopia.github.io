# mlst

**Tags:** <a href="/bactopia-tools/tags/typing" class="badge badge--primary badge--tag">typing</a> <a href="/bactopia-tools/tags/mlst" class="badge badge--primary badge--tag">mlst</a> <a href="/bactopia-tools/tags/sequence-type" class="badge badge--primary badge--tag">sequence-type</a> <a href="/bactopia-tools/tags/alleles" class="badge badge--primary badge--tag">alleles</a> <a href="/bactopia-tools/tags/pubmlst" class="badge badge--primary badge--tag">pubmlst</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

Chamada automática de Multi-Locus tipo de sequencia (MLST) a partir de contigs montados.

Esta Ferramenta Bactopia utiliza o [mlst](https://github.com/tseemann/mlst) para varrer montagens de genomas
e determinar o tipo de sequencia com base nos esquemas do [PubMLST](https://pubmlst.org/). O fluxo de trabalho
detecta automaticamente o esquema MLST adequado para cada organismo e fornece
atribuições de tipo de sequencia padronizadas.

## Uso

Bactopia CLI:

```bash
bactopia --wf mlst \
  --bactopia /path/to/your/bactopia/results
```

Nextflow:

```bash
nextflow run bactopia/bactopia/workflows/bactopia-tools/mlst/main.nf \
  --bactopia /path/to/your/bactopia/results
```

## Parâmetros

### Parâmetros Obrigatórios

Define onde o pipeline deve encontrar os dados de entrada e salvar os dados de saída.

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `--bactopia` | string |  | O caminho para os resultados do bactopia a serem usados como entradas |

### Parâmetros do MLST

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `--mlst_scheme` | string |  | Não detectar automaticamente; forçar este esquema em todas as entradas |
| `--mlst_minid` | integer | `95` | Percentual mínimo de identidade de DNA do alelo completo para ser considerado 'similar' |
| `--mlst_mincov` | integer | `10` | Percentual mínimo de cobertura de DNA para reportar alelo parcial |
| `--mlst_minscore` | integer | `50` | Pontuação mínima de 100 para corresponder a um esquema |
| `--mlst_nopath` | boolean | `false` | Remover caminhos de nome de arquivo da coluna FILE |
| `--mlst_db` | string |  | Um banco de dados MLST personalizado a ser utilizado, como um arquivo compactado ou diretório |
