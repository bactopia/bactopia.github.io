# stecfinder

**Tags:** <a href="/bactopia-tools/tags/stec" class="badge badge--primary badge--tag">stec</a> <a href="/bactopia-tools/tags/serotype" class="badge badge--primary badge--tag">serotype</a> <a href="/bactopia-tools/tags/e-coli" class="badge badge--primary badge--tag">e-coli</a> <a href="/bactopia-tools/tags/shiga-toxin" class="badge badge--primary badge--tag">shiga-toxin</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

Identificação de sorotipo de E. coli produtora de toxina Shiga.

Esta Bactopia Tool utiliza o [STECFinder](https://github.com/LanLab/STECFinder) para identificar
o sorotipo de _E. coli_ produtora de toxina Shiga (STEC) a partir de dados de sequenciamento.
O STECFinder determina o sorotipo, bem como os antígenos O e H.

## Uso

Bactopia CLI:

```bash
bactopia --wf stecfinder \
  --bactopia /path/to/your/bactopia/results
```

Nextflow:

```bash
nextflow run bactopia/bactopia/workflows/bactopia-tools/stecfinder/main.nf \
  --bactopia /path/to/your/bactopia/results
```

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `--stecfinder_use_reads` | boolean | `false` | Reads Illumina paired-end serão usados em vez de montagens |

| `--stecfinder_length` | number | `50.0` | Porcentagem do comprimento do gene necessária para uma chamada positiva |
| `--stecfinder_ipah_length` | number | `10.0` | Porcentagem do comprimento do gene ipaH necessária para uma chamada de gene positiva |
| `--stecfinder_ipah_depth` | number | `1.0` | Profundidade mínima para uma chamada positiva do gene ipaH (requer --stecfinder_use_reads) |
| `--stecfinder_stx_length` | number | `10.0` | Porcentagem do comprimento do gene stx necessária para uma chamada de gene positiva |
| `--stecfinder_stx_depth` | number | `1.0` | Profundidade mínima para uma chamada positiva do gene stx (requer --stecfinder_use_reads) |
| `--stecfinder_o_length` | number | `60.0` | Porcentagem do comprimento do gene wz_ necessária para uma chamada positiva |
