# gtdb

**Tags:** <a href="/bactopia-tools/tags/taxonomy" class="badge badge--primary badge--tag">taxonomy</a> <a href="/bactopia-tools/tags/classification" class="badge badge--primary badge--tag">classification</a> <a href="/bactopia-tools/tags/marker-genes" class="badge badge--primary badge--tag">marker-genes</a> <a href="/bactopia-tools/tags/phylogeny" class="badge badge--primary badge--tag">phylogeny</a> <a href="/bactopia-tools/tags/gtdb" class="badge badge--primary badge--tag">gtdb</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

Identifique genes marcadores e atribua classificações taxonômicas usando GTDB.

Esta Ferramenta Bactopia utiliza o fluxo de trabalho de classificação do [GTDB-Tk](https://github.com/Ecogenomics/GTDBTk)
para atribuir classificações taxonômicas às amostras usando o
[Genome Taxonomy Database](https://gtdb.ecogenomic.org/).

## Uso

Bactopia CLI:

```bash
bactopia --wf gtdb \
  --bactopia /path/to/your/bactopia/results
```

Nextflow:

```bash
nextflow run bactopia/bactopia/workflows/bactopia-tools/gtdb/main.nf \
  --bactopia /path/to/your/bactopia/results
```

## Parâmetros

### Parâmetros Obrigatórios

Define onde o pipeline deve encontrar os dados de entrada e salvar os dados de saída.

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `--bactopia` | string |  | O caminho para os resultados do Bactopia a serem usados como entradas |

### Parâmetros de Configuração do Banco de Dados GTDB-Tk

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `--gtdb` | string |  | Tarball ou caminho de um banco de dados GTDB. Se um banco de dados não for encontrado, você deve usar '--download_gtdb' |
| `--download_gtdb` | boolean | `false` | Baixar o banco de dados GTDB mais recente, mesmo que já exista |
| `--gtdb_save_as_tarball` | boolean | `false` | Baixar o banco de dados GTDB mais recente e salvá-lo em um único tarball |

### Parâmetros de Classificação GTDB

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `--gtdb_min_af` | number | `0.65` | Fração mínima de alinhamento para considerar o genoma mais próximo |
| `--gtdb_min_perc_aa` | integer | `10` | Filtrar genomas com uma porcentagem insuficiente de AA no MSA |
| `--gtdb_tmp` | string | `/tmp` | Especificar diretório alternativo para arquivos temporários |
| `--gtdb_use_scratch` | boolean | `false` | Reduzir o uso de memória do pplacer escrevendo no local --gtdb_tmp (mais lento) |
| `--gtdb_debug` | boolean | `false` | Criar arquivos intermediários para fins de depuração |
| `--gtdb_keep_msa` | boolean | `false` | Manter o arquivo MSA para todos os genomas submetidos e de referência |
| `--force_gtdb` | boolean | `false` | Continuar o processamento se ocorrer um erro em um único genoma |

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `--csvtk_concat_opts` | string |  | Opções extras do csvtk concat entre aspas |
