# traitar

**Tags:** <a href="/bactopia-tools/tags/phenotype" class="badge badge--primary badge--tag">phenotype</a> <a href="/bactopia-tools/tags/traits" class="badge badge--primary badge--tag">traits</a> <a href="/bactopia-tools/tags/pfam" class="badge badge--primary badge--tag">pfam</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

Prever traços fenotípicos a partir de genomas microbianos

Esta Bactopia Tool usa o [Traitar](https://github.com/nick-youngblut/traitar3/) para prever traços fenotípicos a partir de genomas microbianos.

## Uso

Bactopia CLI:

```bash
bactopia --wf traitar \
  --bactopia /path/to/your/bactopia/results
```

Nextflow:

```bash
nextflow run bactopia/bactopia/workflows/bactopia-tools/traitar/main.nf \
  --bactopia /path/to/your/bactopia/results
```

### Resultados por Amostra

| Arquivo | Descrição |
|---------|-----------|
| `*.majority.tsv` | Previsões de traços fenotípicos combinados por votação majoritária |
| `*.single_votes.tsv` | Previsões de traços fenotípicos combinados por voto único |

|-----------|------|--------|-----------|
