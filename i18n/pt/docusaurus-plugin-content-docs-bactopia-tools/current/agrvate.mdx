# agrvate

**Tags:** <a href="/bactopia-tools/tags/staphylococcus-aureus" class="badge badge--primary badge--tag">staphylococcus-aureus</a> <a href="/bactopia-tools/tags/agr" class="badge badge--primary badge--tag">agr</a> <a href="/bactopia-tools/tags/typing" class="badge badge--primary badge--tag">typing</a> <a href="/bactopia-tools/tags/virulence" class="badge badge--primary badge--tag">virulence</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

Identificação rápida do tipo de locus agr de Staphylococcus aureus e variantes do operon agr.

Esta Bactopia Tool utiliza o [AgrVATE](https://github.com/VishnuRaghuram94/AgrVATE) para identificar
rapidamente o tipo de locus _agr_ e detectar variantes do operon agr em montagens de _Staphylococcus aureus_.
O sistema agr é um regulador chave de quorum-sensing da virulência em S. aureus.

## Uso

Bactopia CLI:

```bash
bactopia --wf agrvate \
  --bactopia /path/to/your/bactopia/results
```

Nextflow:

```bash
nextflow run bactopia/bactopia/workflows/bactopia-tools/agrvate/main.nf \
  --bactopia /path/to/your/bactopia/results
```
