# seqsero2

**Tags:** <a href="/bactopia-tools/tags/salmonella" class="badge badge--primary badge--tag">salmonella</a> <a href="/bactopia-tools/tags/serotyping" class="badge badge--primary badge--tag">serotyping</a> <a href="/bactopia-tools/tags/epidemiology" class="badge badge--primary badge--tag">epidemiology</a> <a href="/bactopia-tools/tags/o-antigen" class="badge badge--primary badge--tag">o-antigen</a> <a href="/bactopia-tools/tags/h-antigen" class="badge badge--primary badge--tag">h-antigen</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

Predição de sorotipo de Salmonella a partir de reads de sequenciamento ou montagens.

Esta Ferramenta Bactopia utiliza o [SeqSero2](https://github.com/denglab/SeqSero2) para prever
sorotipos de Salmonella tanto a partir de reads brutos de sequenciamento quanto de genomas montados.
SeqSero2 é um pipeline inovador para determinar sorotipos de Salmonella usando reads brutos de
sequenciamento ou montagens, por meio de análise de k-mer e identificação direcionada dos genes
dos antígenos O e H. A ferramenta fornece predições precisas de sorotipo seguindo o esquema
Kaufmann-White, suportando métodos de sorotipagem tradicionais e moleculares para vigilância
epidemiológica e investigação de surtos.

## Uso

Bactopia CLI:

```bash
bactopia --wf seqsero2 \
  --bactopia /path/to/your/bactopia/results
```

Nextflow:

```bash
nextflow run bactopia/bactopia/workflows/bactopia-tools/seqsero2/main.nf \
  --bactopia /path/to/your/bactopia/results
```
