# staphscan

**Tags:** <a href="/bactopia-tools/tags/staphylococcus-aureus" class="badge badge--primary badge--tag">staphylococcus-aureus</a> <a href="/bactopia-tools/tags/surveillance" class="badge badge--primary badge--tag">surveillance</a> <a href="/bactopia-tools/tags/mlst" class="badge badge--primary badge--tag">mlst</a> <a href="/bactopia-tools/tags/spa-typing" class="badge badge--primary badge--tag">spa-typing</a> <a href="/bactopia-tools/tags/sccmec" class="badge badge--primary badge--tag">sccmec</a> <a href="/bactopia-tools/tags/amr" class="badge badge--primary badge--tag">amr</a> <a href="/bactopia-tools/tags/virulence" class="badge badge--primary badge--tag">virulence</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

Análise de vigilância genômica de Staphylococcus aureus.

Esta Bactopia Tool utiliza o [StaphSCAN](https://github.com/riccabolla/StaphSCAN) para realizar
vigilância genômica de _Staphylococcus aureus_ para tipagem epidemiológica e
perfil de resistência.

## Uso

Bactopia CLI:

```bash
bactopia --wf staphscan \
  --bactopia /path/to/your/bactopia/results
```

Nextflow:

```bash
nextflow run bactopia/bactopia/workflows/bactopia-tools/staphscan/main.nf \
  --bactopia /path/to/your/bactopia/results
```
