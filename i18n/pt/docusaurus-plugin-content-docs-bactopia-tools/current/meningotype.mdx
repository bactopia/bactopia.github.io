# meningotype

**Tags:** <a href="/bactopia-tools/tags/neisseria-meningitidis" class="badge badge--primary badge--tag">neisseria-meningitidis</a> <a href="/bactopia-tools/tags/serotyping" class="badge badge--primary badge--tag">serotyping</a> <a href="/bactopia-tools/tags/mlst" class="badge badge--primary badge--tag">mlst</a> <a href="/bactopia-tools/tags/finetyping" class="badge badge--primary badge--tag">finetyping</a> <a href="/bactopia-tools/tags/fasta" class="badge badge--primary badge--tag">fasta</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

Tipagem abrangente de Neisseria meningitidis.

Esta Bactopia Tool utiliza o [meningotype](https://github.com/MDU-PHL/meningotype)
para tipagem _in silico_ de genomas de _Neisseria meningitidis_. Ele usa contigs de
montagem para determinar o sorotipo, MLST, finetyping (_porA_, _fetA_, _porB_) e
tipagem de sequência de antígeno Bexsero (BAST) (_fHbp_, _NHBA_, _NadA_, _PorA_).

## Uso

Bactopia CLI:

```bash
bactopia --wf meningotype \
  --bactopia /path/to/your/bactopia/results
```

Nextflow:

```bash
nextflow run bactopia/bactopia/workflows/bactopia-tools/meningotype/main.nf \
  --bactopia /path/to/your/bactopia/results
```

Você pode usar estes parâmetros para ajustar sua análise com o meningotype

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `--meningotype_finetype` | boolean | `false` | realiza finetyping de porA e fetA |
| `--meningotype_porB` | boolean | `false` | realiza tipagem de sequência porB (NEIS2020) |
| `--meningotype_bast` | boolean | `false` | realiza tipagem de sequência de antígeno Bexsero (BAST) |
| `--meningotype_mlst` | boolean | `false` | realiza MLST |
| `--meningotype_all` | boolean | `false` | realiza tipagem MLST, porA, fetA, porB, BAST |
