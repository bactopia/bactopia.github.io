# teton

**Tags:** <a href="/bactopia-pipelines/tags/metagenomics" class="badge badge--primary badge--tag">metagenomics</a> <a href="/bactopia-pipelines/tags/classification" class="badge badge--primary badge--tag">classification</a> <a href="/bactopia-pipelines/tags/kraken2" class="badge badge--primary badge--tag">kraken2</a> <a href="/bactopia-pipelines/tags/bracken" class="badge badge--primary badge--tag">bracken</a> <a href="/bactopia-pipelines/tags/abundance" class="badge badge--primary badge--tag">abundance</a> <a href="/bactopia-pipelines/tags/profiling" class="badge badge--primary badge--tag">profiling</a> <a href="/bactopia-pipelines/tags/named-workflow" class="badge badge--primary badge--tag">named-workflow</a> 

Taxonomic classification and abundance profiling of metagenomic reads.

This workflow performs metagenomic classification using [Kraken2](https://github.com/DerrickWood/kraken2)
and [Bracken](https://github.com/jenniferlu717/Bracken), with optional host read removal
using [deacon](https://github.com/bede/deacon) (default), [nohuman](https://github.com/mbhall88/nohuman),
or [SRA Human Scrubber](https://github.com/ncbi/sra-human-scrubber). It processes metagenomic
sequencing reads to estimate bacterial genome sizes and separate bacterial from non-bacterial organisms.

## Usage

teton CLI:

```bash
teton \
  --input samples.csv \
  --outdir results/
```

Nextflow:

```bash
nextflow run bactopia/bactopia/workflows/teton/main.nf \
  --input samples.csv \
  --outdir results/
```

## Composition

This workflow uses the following subworkflows:

- [bactopia_gather](/developers/subworkflows/bactopia_gather) - Search, validate, gather, and standardize input samples.
- [teton](/developers/subworkflows/teton) - Perform taxonomic classification and estimate bacterial genome sizes.
