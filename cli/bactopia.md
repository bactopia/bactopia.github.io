# bactopia

The main entry point for running Bactopia workflows.

## Usage

```bash
bactopia [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--input` | Path to input FASTQ files |
| `--datasets` | Path to pre-built datasets |
| `--species` | Species name for auto-download |
