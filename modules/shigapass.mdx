# shigapass

<span class="badge badge--success">stable</span> <span class="badge badge--secondary">shigapass v1.5.0</span> <span class="badge badge--info">sample scope</span>

Predict Shigella serotypes and differentiate Shigella/EIEC.

Uses [ShigaPass](https://github.com/Munch-Lab/ShigaPass) to identify *Shigella* serotypes and
distinguish *Shigella* species from Enteroinvasive *Escherichia coli* (EIEC) using specific
genomic markers from assembled contigs.

## Inputs

```
record (
    meta: Record,
    fna: Path
)
```

| Field | Type | Description |
|-------|------|-------------|
| `meta` | `Record` | Groovy Record containing sample information |
| `fna` | `Path` | Assembled contigs in FASTA format |

## Outputs

```
record (
    meta: Record,
    tsv: Path,
    flex_tsv: Path?,
    results: Set<Path>,
    logs: Set<Path?>,
    nf_logs: Set<Path>,
    versions: Set<Path>
)
```

| Field | Type | Description |
|-------|------|-------------|
| `meta` | `Record` | Sample information record |
| `tsv` | `Path` | ShigaPass summary results in TSV format |
| `flex_tsv` | `Path?` | ShigaPass Flex summary results in TSV format |
| `results` | `Set<Path>` | All output files to be published |
| `logs` | `Set<Path?>` | Optional program specific log files |
| `nf_logs` | `Set<Path>` | Nextflow-specific log files (e.g. .command.&#123;begin|err|log|out|run|sh|trace&#125;) |
| `versions` | `Set<Path>` | A YAML formatted file with program versions |
