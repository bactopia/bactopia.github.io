# bactopia_datasets

<span class="badge badge--success">stable</span> <span class="badge badge--secondary">gnu-wget v1.18</span> <span class="badge badge--info">run scope</span>

Download pre-compiled datasets required by Bactopia.

Fetches the core datasets (AMR, MLST, Mash, Sourmash) hosted by the Bactopia project.
These are used to populate the local cache for offline use.

:::note[Internet Required]
This process requires an active internet connection to fetch files from `datasets.bactopia.com`.
:::

## Outputs

```
record (
    amrfinderplus_db: Path,
    mlst_db: Path,
    mash_db: Path,
    sourmash_db: Path
)
```

| Field | Type | Description |
|-------|------|-------------|
| `amrfinderplus_db` | `Path` | A compressed tarball of the [AMRFinderPlus](https://github.com/ncbi/amr) database |
| `mlst_db` | `Path` | A compressed tarball of the [PubMLST](https://pubmlst.org/) schemes |
| `mash_db` | `Path` | Pre-computed [Mash](https://github.com/marbl/Mash) sketches (RefSeq) |
| `sourmash_db` | `Path` | Pre-computed [Sourmash](https://github.com/sourmash-bio/sourmash) signatures (GTDB) |
