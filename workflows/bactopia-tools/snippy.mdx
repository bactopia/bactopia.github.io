# snippy

<span class="badge badge--success">stable</span>

Rapid haplotype variant calling and core genome alignment.

This Bactopia Tool uses [Snippy](https://github.com/tseemann/snippy) to find SNPs between a
reference genome and a set of reads, perform core genome alignment, and generate
phylogenetic trees. It includes optional recombination detection with Gubbins
and phylogenetic tree construction with IQ-Tree.

## Usage

```bash
bactopia --wf snippy \
  --bactopia /path/to/your/bactopia/results
```

|-----------|------|---------|-------------|
| `--species` | string |  | Name of the species to download assemblies |
| `--accession` | string |  | An NCBI Assembly accession to be downloaded |

## Composition

This workflow uses the following subworkflows:

- [gubbins](/subworkflows/gubbins) - Detect and filter recombination regions in bacterial alignments.
- [iqtree](/subworkflows/iqtree) - Construct maximum likelihood phylogenetic trees from alignments.
- [ncbigenomedownload](/subworkflows/ncbigenomedownload) - Download bacterial genomes from NCBI's RefSeq database.
- [snippy_core](/subworkflows/snippy_core) - Generate core-genome SNP alignment from per-sample Snippy outputs.
- [snippy_run](/subworkflows/snippy_run) - Call variants against a reference genome using Snippy.
