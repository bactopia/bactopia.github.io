# clermontyping

<span class="badge badge--success">stable</span>

In silico phylotyping of Escherichia genus.

This Bactopia Tool uses [ClermonTyping](https://github.com/happykhan/ClermonTyping)
to conduct _in silico_ prediction of phylotype for _Escherichia_ genomes. It uses
genome assemblies to assign them to _E. albertii_, _E. fergusonii_, _Escherichia_
clades I–V, _E. coli sensu stricto_ as well as to the main _E. coli_ phylogroups.

## Usage

```bash
bactopia --wf clermontyping \
  --bactopia /path/to/your/bactopia/results
```

### Per-Sample Results

| File | Description |
|------|-------------|
| `*.blast.xml` | BLAST XML file with ClermonTyping analysis results |
| `*.html` | HTML file with ClermonTyping analysis results |
| `*.mash.tsv` | TSV file with Mash distances |
| `*.phylogroups.txt` | TSV file with final phylogroup assignments |

### Merged Results

| File | Description |
|------|-------------|
| `clermontyping.tsv` | Merged TSV file with ClermonTyping results from all samples |
