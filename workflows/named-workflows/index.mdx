# Named Workflows

Named workflows are complete analysis pipelines that process raw sequencing data through multiple analysis steps. There are 4 named workflows available.

| Workflow | Status | Description |
|----------|--------|-------------|
| [bactopia](/workflows/named-workflows/bactopia) | stable | Comprehensive bacterial analysis pipeline for complete genomic characterization. |
| [cleanyerreads](/workflows/named-workflows/cleanyerreads) | stable | Quality control and optional host read removal from raw sequencing reads. |
| [staphopia](/workflows/named-workflows/staphopia) | stable | Comprehensive analysis pipeline for Staphylococcus aureus isolates. |
| [teton](/workflows/named-workflows/teton) | stable | Taxonomic classification and abundance profiling of metagenomic reads. |
