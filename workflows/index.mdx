# Workflows

Bactopia includes 70 workflows: 4 named workflows and 66 tool workflows.

## Named Workflows

There are 4 Named Workflows that are complete analysis workflows processing raw sequencing data through multiple analysis steps.

| Workflow | Description |
|----------|-------------|
| [bactopia](/workflows/named-workflows/bactopia) | Comprehensive bacterial analysis pipeline for complete genomic characterization. |
| [cleanyerreads](/workflows/named-workflows/cleanyerreads) | Quality control and optional host read removal from raw sequencing reads. |
| [staphopia](/workflows/named-workflows/staphopia) | Comprehensive analysis pipeline for Staphylococcus aureus isolates. |
| [teton](/workflows/named-workflows/teton) | Taxonomic classification and abundance profiling of metagenomic reads. |
