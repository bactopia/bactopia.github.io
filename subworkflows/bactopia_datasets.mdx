# bactopia_datasets

<span class="badge badge--success">stable</span> <span class="badge badge--info">custom scope</span>

Download and provide pre-compiled datasets required by Bactopia.

This subworkflow wraps the DATASETS module and extracts individual database
paths as separate channel emissions for downstream consumption.

## Emit

### Published

The `sample_outputs` and `run_outputs` emissions are aggregates of output files that will be published in the entry workflow.

### Downstream Inputs

The following emissions are meant to be used as inputs to downstream subworkflows.

#### `amrfinderplus_db`

Path to the AMRFinderPlus database tarball

#### `mlst_db`

Path to the PubMLST database tarball

#### `mash_db`

Path to the Mash RefSeq sketch

#### `sourmash_db`

Path to the Sourmash GTDB signatures

## Module Composition

This subworkflow calls the following modules:

- [bactopia_datasets](/modules/bactopia_datasets) - Download pre-compiled datasets required by Bactopia.

## Used By

This subworkflow is used by the following workflows:

- [amrfinderplus](/workflows/bactopia-tools/amrfinderplus) - Bactopia Tool: Amrfinderplus.
- [bactopia](/workflows/named-workflows/bactopia) - Comprehensive bacterial analysis pipeline for complete genomic characterization.
- [merlin](/workflows/bactopia-tools/merlin) - MinMER-assisted species-specific tool selection and execution.
- [staphopia](/workflows/named-workflows/staphopia) - Comprehensive analysis pipeline for Staphylococcus aureus isolates.
