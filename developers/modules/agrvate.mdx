# agrvate

**Tags:** <a href="/developers/tags/bacteria" class="badge badge--primary badge--tag">bacteria</a> <a href="/developers/tags/assembly" class="badge badge--primary badge--tag">assembly</a> <a href="/developers/tags/fasta" class="badge badge--primary badge--tag">fasta</a> <a href="/developers/tags/typing" class="badge badge--primary badge--tag">typing</a> <a href="/developers/tags/virulence" class="badge badge--primary badge--tag">virulence</a> <a href="/developers/tags/staphylococcus" class="badge badge--primary badge--tag">staphylococcus</a> <a href="/developers/tags/aureus" class="badge badge--primary badge--tag">aureus</a> <a href="/developers/tags/agr" class="badge badge--primary badge--tag">agr</a> <a href="/developers/tags/sample-scope" class="badge badge--primary badge--tag">sample-scope</a> 

Determine the agr locus type and operon variants in Staphylococcus aureus.

Uses [AgrVATE](https://github.com/VishnuRaghuram94/AgrVATE) to type the accessory gene
regulator (agr) locus, a quorum sensing system critical for *Staphylococcus aureus* virulence.

## Inputs

```
record (
    meta: Record,
    fna: Path
)
```

| Field | Type | Description |
|-------|------|-------------|
| `meta` | `Record` | Groovy Record containing sample information |
| `fna` | `Path` | Assembled Staphylococcus aureus contigs in FASTA format |

## Outputs

```
record (
    meta: Record,
    tsv: Path,
    results: Set<Path>,
    logs: Set<Path?>,
    nf_logs: Set<Path>,
    versions: Set<Path>
)
```

| Field | Type | Description |
|-------|------|-------------|
| `meta` | `Record` | Sample information record |

| `results` | `Set<Path>` | All output files to be published |
| `logs` | `Set<Path?>` | Optional program specific log files |
| `nf_logs` | `Set<Path>` | Nextflow-specific log files (e.g. .command.&#123;begin|err|log|out|run|sh|trace&#125;) |
| `versions` | `Set<Path>` | A YAML formatted file with program versions |

## Parameters

### AgrVATE Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `--agrvate_typing_only` | boolean | `false` | agr typing only. Skips agr operon extraction and frameshift detection |
