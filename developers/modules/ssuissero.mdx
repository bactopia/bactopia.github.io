# ssuissero

**Tags:** <a href="/developers/tags/streptococcus-suis" class="badge badge--primary badge--tag">streptococcus-suis</a> <a href="/developers/tags/serotype" class="badge badge--primary badge--tag">serotype</a> <a href="/developers/tags/typing" class="badge badge--primary badge--tag">typing</a> <a href="/developers/tags/prediction" class="badge badge--primary badge--tag">prediction</a> <a href="/developers/tags/sample-scope" class="badge badge--primary badge--tag">sample-scope</a> 

Serotype prediction of Streptococcus suis assemblies.

Uses [SsuisSero](https://github.com/idmc-cnr/SsuisSero) to predict the serotype of
*Streptococcus suis* strains from genome assemblies based on the presence of specific
capsular genes.

## Inputs

```
record (
    meta: Record,
    fna: Path
)
```

| Field | Type | Description |
|-------|------|-------------|
| `meta` | `Record` | Groovy Record containing sample information |
| `fna` | `Path` | Assembled contigs in FASTA format |

## Outputs

```
record (
    meta: Record,
    tsv: Path,
    results: Set<Path>,
    logs: Set<Path?>,
    nf_logs: Set<Path>,
    versions: Set<Path>
)
```

| Field | Type | Description |
|-------|------|-------------|
| `meta` | `Record` | Sample information record |
| `tsv` | `Path` | SsuisSero results in TSV format |
| `results` | `Set<Path>` | All output files to be published |
| `logs` | `Set<Path?>` | Optional program specific log files |
| `nf_logs` | `Set<Path>` | Nextflow-specific log files (e.g. .command.&#123;begin|err|log|out|run|sh|trace&#125;) |
| `versions` | `Set<Path>` | A YAML formatted file with program versions |
