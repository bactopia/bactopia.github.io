# spatyper

**Tags:** <a href="/developers/tags/staphylococcus-aureus" class="badge badge--primary badge--tag">staphylococcus-aureus</a> <a href="/developers/tags/spa-typing" class="badge badge--primary badge--tag">spa-typing</a> <a href="/developers/tags/repeat" class="badge badge--primary badge--tag">repeat</a> <a href="/developers/tags/mrsa" class="badge badge--primary badge--tag">mrsa</a> <a href="/developers/tags/typing" class="badge badge--primary badge--tag">typing</a> <a href="/developers/tags/sample-scope" class="badge badge--primary badge--tag">sample-scope</a> 

Finding spa types in Staphylococcus aureus.

Uses [spaTyper](https://github.com/HCGB-IGTP/spaTyper) to determine the *spa* type of
*Staphylococcus aureus* genomes by identifying the repeats in the polymorphic X region
of the protein A gene (*spa*).

## Inputs

```
record (
    meta: Record,
    fna: Path
)
```

| Field | Type | Description |
|-------|------|-------------|
| `meta` | `Record` | Groovy Record containing sample information |
| `fna` | `Path` | Assembled contigs in FASTA format |

```
repeats: Path?
repeat_order: Path?
```

| Name | Type | Description |
|------|------|-------------|
| `repeats` | `Path?` | Custom repeat sequences file |
| `repeat_order` | `Path?` | Custom repeat order file |

## Outputs

```
record (
    meta: Record,
    tsv: Path,
    results: Set<Path>,
    logs: Set<Path?>,
    nf_logs: Set<Path>,
    versions: Set<Path>
)
```

| Field | Type | Description |
|-------|------|-------------|
| `meta` | `Record` | Sample information record |
| `tsv` | `Path` | spa typing results in TSV format |
| `results` | `Set<Path>` | All output files to be published |
| `logs` | `Set<Path?>` | Optional program specific log files |
| `nf_logs` | `Set<Path>` | Nextflow-specific log files (e.g. .command.&#123;begin|err|log|out|run|sh|trace&#125;) |
| `versions` | `Set<Path>` | A YAML formatted file with program versions |
