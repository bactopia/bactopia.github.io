# defensefinder_update

**Tags:** <a href="/developers/tags/bacteria" class="badge badge--primary badge--tag">bacteria</a> <a href="/developers/tags/defense" class="badge badge--primary badge--tag">defense</a> <a href="/developers/tags/database" class="badge badge--primary badge--tag">database</a> <a href="/developers/tags/download" class="badge badge--primary badge--tag">download</a> <a href="/developers/tags/hmm" class="badge badge--primary badge--tag">hmm</a> <a href="/developers/tags/casfinder" class="badge badge--primary badge--tag">casfinder</a> <a href="/developers/tags/crispr" class="badge badge--primary badge--tag">crispr</a> <a href="/developers/tags/run-scope" class="badge badge--primary badge--tag">run-scope</a> 

Download and package the DefenseFinder and CasFinder model databases.

Fetches the latest HMM profiles from the [DefenseFinder](https://github.com/mdmparis/defense-finder-models)
and [CasFinder](https://github.com/macsy-models/CasFinder) repositories, then packages them
into a single tarball for use by the DefenseFinder module.

:::note[Internet Required]
This process requires an active internet connection to fetch the models from GitHub.
:::

## Outputs

```
record (
    db: Path,
    logs: Set<Path?>
)
```

| Field | Type | Description |
|-------|------|-------------|
| `db` | `Path` | A compressed tarball containing both DefenseFinder and CasFinder models |
| `logs` | `Set<Path?>` | Optional program specific log files |
