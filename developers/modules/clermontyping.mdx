# clermontyping

**Tags:** <a href="/developers/tags/bacteria" class="badge badge--primary badge--tag">bacteria</a> <a href="/developers/tags/escherichia-coli" class="badge badge--primary badge--tag">escherichia-coli</a> <a href="/developers/tags/typing" class="badge badge--primary badge--tag">typing</a> <a href="/developers/tags/phylotyping" class="badge badge--primary badge--tag">phylotyping</a> <a href="/developers/tags/pcr" class="badge badge--primary badge--tag">pcr</a> <a href="/developers/tags/phylogroup" class="badge badge--primary badge--tag">phylogroup</a> <a href="/developers/tags/sample-scope" class="badge badge--primary badge--tag">sample-scope</a> 

Determine the phylogroup of Escherichia coli isolates.

Uses [ClermonTyping](https://github.com/A-BN/ClermonTyping) to perform in silico PCR
detection of specific marker genes (arpA, chuA, yjaA, TspE4.C2). This assigns the
isolate to one of the main *E. coli* phylogroups (A, B1, B2, C, D, E, F, G, or Cryptic).

## Inputs

```
record (
    meta: Record,
    fna: Path
)
```

| Field | Type | Description |
|-------|------|-------------|
| `meta` | `Record` | Groovy Record containing sample information |
| `fna` | `Path` | Assembled contigs in FASTA format |
