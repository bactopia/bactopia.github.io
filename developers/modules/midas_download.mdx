# midas_download

**Tags:** <a href="/developers/tags/midas" class="badge badge--primary badge--tag">midas</a> <a href="/developers/tags/download" class="badge badge--primary badge--tag">download</a> <a href="/developers/tags/database" class="badge badge--primary badge--tag">database</a> <a href="/developers/tags/metagenomics" class="badge badge--primary badge--tag">metagenomics</a> <a href="/developers/tags/species" class="badge badge--primary badge--tag">species</a> <a href="/developers/tags/run-scope" class="badge badge--primary badge--tag">run-scope</a> 

Download the MIDAS reference database.

Fetches the pre-compiled database required by [MIDAS](https://github.com/snayfach/MIDAS)
for metagenomic species profiling. The database contains reference genomes from the
UHGG collection used for species identification and abundance estimation.

:::note[Internet & Storage Required]
This process requires an active internet connection and significant disk space
to store the database files (~3.3GB compressed, ~7GB uncompressed).
:::

## Outputs

```
record (
    db: Path?,
    db_tarball: Path?,
    logs: Set<Path?>
)
```

| Field | Type | Description |
|-------|------|-------------|
| `db` | `Path?` | The MIDAS database directory containing reference genome data |
| `db_tarball` | `Path?` | A compressed tarball of the database (if requested via parameters) |
| `logs` | `Set<Path?>` | Optional program specific log files |

## Parameters

### MIDAS Database Download Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `--midas_db` | string |  | A single tarball or path to a MIDAS formatted database |
| `--midas_save_as_tarball` | boolean | `false` | Save the MIDAS database as a tarball |
| `--download_midas` | boolean | `false` | Download the MIDAS database to the path given by --midas_db |
