# staphopiasccmec

**Tags:** <a href="/developers/tags/staphylococcus-aureus" class="badge badge--primary badge--tag">staphylococcus-aureus</a> <a href="/developers/tags/sccmec" class="badge badge--primary badge--tag">sccmec</a> <a href="/developers/tags/typing" class="badge badge--primary badge--tag">typing</a> <a href="/developers/tags/mrsa" class="badge badge--primary badge--tag">mrsa</a> <a href="/developers/tags/primers" class="badge badge--primary badge--tag">primers</a> <a href="/developers/tags/sample-scope" class="badge badge--primary badge--tag">sample-scope</a> 

Primer based SCCmec typing of S. aureus genomes.

Uses [Staphopia SCCmec](https://github.com/staphopia/staphopia-sccmec) to determine the
SCCmec type of *Staphylococcus aureus* assemblies. It includes a primer-based approach
to identify SCCmec types I-XI.

## Inputs

```
record (
    meta: Record,
    fna: Path
)
```

| Field | Type | Description |
|-------|------|-------------|
| `meta` | `Record` | Groovy Record containing sample information |
| `fna` | `Path` | Assembled contigs in FASTA format |

record (
    meta: Record,

| Field | Type | Description |

| `nf_logs` | `Set<Path>` | Nextflow-specific log files (e.g. .command.&#123;begin|err|log|out|run|sh|trace&#125;) |

## Parameters

### Staphopia SCCmec Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `--staphopiasccmec_hamming` | boolean | `false` | Report the results as hamming distances |
