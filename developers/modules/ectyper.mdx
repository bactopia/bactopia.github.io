# ectyper

**Tags:** <a href="/developers/tags/bacteria" class="badge badge--primary badge--tag">bacteria</a> <a href="/developers/tags/escherichia-coli" class="badge badge--primary badge--tag">escherichia-coli</a> <a href="/developers/tags/serotype" class="badge badge--primary badge--tag">serotype</a> <a href="/developers/tags/o-antigen" class="badge badge--primary badge--tag">o-antigen</a> <a href="/developers/tags/h-antigen" class="badge badge--primary badge--tag">h-antigen</a> <a href="/developers/tags/typing" class="badge badge--primary badge--tag">typing</a> <a href="/developers/tags/sample-scope" class="badge badge--primary badge--tag">sample-scope</a> 

Predict *Escherichia coli* serotype (O and H antigens).

Uses [ECTyper](https://github.com/phac-nml/ectyper) to identify the O-antigen (lipopolysaccharide)
and H-antigen (flagella) genes in *E. coli* genome assemblies via BLAST. It provides a
standardized serotype prediction (e.g., O157:H7).

## Inputs

```
record (
    meta: Record,
    fna: Path
)
```

| Field | Type | Description |
|-------|------|-------------|
| `meta` | `Record` | Groovy Record containing sample information |
| `fna` | `Path` | Assembled contigs in FASTA format |

## Outputs

```
record (
    meta: Record,
    tsv: Path,
    txt: Path,
    results: Set<Path>,
    logs: Set<Path?>,
    nf_logs: Set<Path>,
    versions: Set<Path>
)
```

| Field | Type | Description |
|-------|------|-------------|
| `meta` | `Record` | Sample information record |
| `tsv` | `Path` | Tab-delimited E. coli serotype predictions (O and H antigens) |
| `txt` | `Path` | BLAST allele details for serotype determination |
| `results` | `Set<Path>` | All output files to be published |
| `logs` | `Set<Path?>` | Optional program specific log files |
| `nf_logs` | `Set<Path>` | Nextflow-specific log files (e.g. .command.&#123;begin|err|log|out|run|sh|trace&#125;) |
| `versions` | `Set<Path>` | A YAML formatted file with program versions |

## Parameters

### ECTyper Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `--ectyper_opid` | integer | `90` | Percent identity required for an O antigen allele match |
| `--ectyper_opcov` | integer | `90` | Minimum percent coverage required for an O antigen allele match |
| `--ectyper_hpid` | integer | `95` | Percent identity required for an H antigen allele match |
| `--ectyper_hpcov` | integer | `50` | Minimum percent coverage required for an H antigen allele match |
