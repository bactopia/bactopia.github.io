# pasty

**Tags:** <a href="/developers/tags/pseudomonas-aeruginosa" class="badge badge--primary badge--tag">pseudomonas-aeruginosa</a> <a href="/developers/tags/serogroup" class="badge badge--primary badge--tag">serogroup</a> <a href="/developers/tags/typing" class="badge badge--primary badge--tag">typing</a> <a href="/developers/tags/o-antigen" class="badge badge--primary badge--tag">o-antigen</a> <a href="/developers/tags/prediction" class="badge badge--primary badge--tag">prediction</a> <a href="/developers/tags/sample-scope" class="badge badge--primary badge--tag">sample-scope</a> 

Predict serogroups of Pseudomonas aeruginosa from assemblies.

This subworkflow uses [Pasty](https://github.com/rpetit3/pasty) to perform in silico
serogrouping of *Pseudomonas aeruginosa* isolates from assembled genomes. It identifies
O-antigen biosynthesis genes to classify isolates into their known serogroups using
BLAST-based homology searches.

## Take

```
assembly: Channel<Record>
```

| Field | Description |
|-------|-------------|
| `meta` | Groovy Record containing sample information |
| `assembly` | Assembled contigs in FASTA format |

## Emit

### Published

The `sample_outputs` and `run_outputs` emissions are aggregates of output files that will be published in the entry workflow.

#### `sample_outputs`

| Output | Description |
|--------|-------------|
| `tsv` | A tab-delimited summary file with the predicted O-antigen serogroup |
| `blast` | A tab-delimited file of all raw BLAST hits used for the prediction |
| `details` | A tab-delimited file with detailed gene hits for each serogroup tested |

#### `run_outputs`

| Output | Description |
|--------|-------------|
| `csv` | Aggregated results in CSV format |
