# scoary

**Tags:** <a href="/developers/tags/gwas" class="badge badge--primary badge--tag">gwas</a> <a href="/developers/tags/association" class="badge badge--primary badge--tag">association</a> <a href="/developers/tags/pan-genome" class="badge badge--primary badge--tag">pan-genome</a> <a href="/developers/tags/traits" class="badge badge--primary badge--tag">traits</a> <a href="/developers/tags/statistical" class="badge badge--primary badge--tag">statistical</a> <a href="/developers/tags/run-scope" class="badge badge--primary badge--tag">run-scope</a> 

Pan-genome wide association studies.

This subworkflow performs genome-wide association studies (GWAS) on pan-genome data
using [Scoary](https://github.com/AdmiralenOla/Scoary). The tool identifies genes
associated with binary traits such as pathogenicity, host specificity, or antibiotic
resistance. It calculates statistical associations between gene presence/absence
and phenotypic traits across multiple bacterial isolates.

## Take

```
csv: Channel<Record>
```

| Field | Description |
|-------|-------------|
| `meta` | Groovy Record containing sample information |
| `csv` | Gene presence/absence matrix from pan-genome analysis in CSV format |

```
traits: Path?
```

| Name | Type | Description |
|------|------|-------------|
| `traits` | `Path?` | Trait file containing binary phenotypic characteristics for each isolate (optional) |
