# blastn

**Tags:** <a href="/developers/tags/blast" class="badge badge--primary badge--tag">blast</a> <a href="/developers/tags/alignment" class="badge badge--primary badge--tag">alignment</a> <a href="/developers/tags/nucleotide" class="badge badge--primary badge--tag">nucleotide</a> <a href="/developers/tags/search" class="badge badge--primary badge--tag">search</a> <a href="/developers/tags/fasta" class="badge badge--primary badge--tag">fasta</a> <a href="/developers/tags/database" class="badge badge--primary badge--tag">database</a> <a href="/developers/tags/sample-scope" class="badge badge--primary badge--tag">sample-scope</a> 

Search a nucleotide database using nucleotide query sequences.

This subworkflow uses [BLASTN](https://blast.ncbi.nlm.nih.gov/Blast.cgi) to align
nucleotide query sequences against a nucleotide BLAST database. It processes each
sample individually and aggregates the alignment results into a single consolidated
report for all samples.

## Take

```
blastdb: Channel<Record>
```

| Field | Description |
|-------|-------------|
| `meta` | Groovy Record containing sample information |
| `blastdb` | A compressed tarball containing the BLAST database |

```
query: Path
```

| Name | Type | Description |
|------|------|-------------|
| `query` | `Path` | FASTA file containing nucleotide query sequences to search for in the assemblies |

## Emit

### Published

The `sample_outputs` and `run_outputs` emissions are aggregates of output files that will be published in the entry workflow.

#### `sample_outputs`

| Output | Description |
|--------|-------------|
| `tsv` | A tab-delimited summary of alignments (standard BLAST outfmt 6) |

#### `run_outputs`

| Output | Description |
|--------|-------------|
| `csv` | Aggregated results in CSV format |

## Module Composition

This subworkflow calls the following modules:

- [csvtk_concat](/developers/modules/csvtk_concat) - Concatenate multiple CSV or TSV files into a single table.
