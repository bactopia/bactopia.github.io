# bactopia-check-assembly-accession

Verify NCBI Assembly accession is latest and still available.

## Usage

```bash
bactopia-check-assembly-accession REFERENCE [OPTIONS]
```

## Arguments

| Argument | Type | Required | Description |
|----------|------|----------|-------------|
| `REFERENCE` | STRING | Yes | The assembly accession to verify. |

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `--version, -V` | BOOL | `false` | Show the version and exit. |
