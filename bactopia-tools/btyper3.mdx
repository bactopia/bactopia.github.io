# btyper3

**Tags:** <a href="/bactopia-tools/tags/bacillus-cereus" class="badge badge--primary badge--tag">bacillus-cereus</a> <a href="/bactopia-tools/tags/taxonomy" class="badge badge--primary badge--tag">taxonomy</a> <a href="/bactopia-tools/tags/classification" class="badge badge--primary badge--tag">classification</a> <a href="/bactopia-tools/tags/fasta" class="badge badge--primary badge--tag">fasta</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

Taxonomic classification of Bacillus cereus group isolates.

This Bactopia Tool uses [BTyper3](https://github.com/lmc297/BTyper3) to classify
Bacillus cereus group isolates from genome assemblies.

## Usage

Bactopia CLI:

```bash
bactopia --wf btyper3 \
  --bactopia /path/to/your/bactopia/results
```

Nextflow:

```bash
nextflow run bactopia/bactopia/workflows/bactopia-tools/btyper3/main.nf \
  --bactopia /path/to/your/bactopia/results
```

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `--btyper3_virulence_identity` | integer | `70` | Minimum percent amino acid/nucleotide identity threshold for a virulence gene to be considered present |

| `--btyper3_identity` | integer | `50` | Minimum percent amino acid identity threshold for a Bt toxin gene to be considered present |
| `--btyper3_coverage` | integer | `70` | Minimum percent coverage threshold for a Bt toxin gene to be considered present |
| `--btyper3_overlap` | number | `0.7` | Specify maximum proportion of overlap for overlapping Bt toxin genes to be considered separate genes |
| `--btyper3_opts` | string |  | Additional options to pass to BTyper3 |
