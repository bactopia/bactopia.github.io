# sistr

**Tags:** <a href="/bactopia-tools/tags/salmonella" class="badge badge--primary badge--tag">salmonella</a> <a href="/bactopia-tools/tags/serovar" class="badge badge--primary badge--tag">serovar</a> <a href="/bactopia-tools/tags/cgmlst" class="badge badge--primary badge--tag">cgmlst</a> <a href="/bactopia-tools/tags/typing" class="badge badge--primary badge--tag">typing</a> <a href="/bactopia-tools/tags/sistr" class="badge badge--primary badge--tag">sistr</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

Serovar prediction of Salmonella enterica from assemblies.

This Bactopia Tool uses [Salmonella In Silico Typing Resource](https://github.com/phac-nml/sistr_cmd),
or SISTR, for serovar prediction of Salmonella enterica assemblies using cgMLST typing
and molecular serovar prediction.

## Usage

```bash
bactopia --wf sistr \
  --bactopia /path/to/your/bactopia/results
```
