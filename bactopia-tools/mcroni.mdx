# mcroni

**Tags:** <a href="/bactopia-tools/tags/mcr-1" class="badge badge--primary badge--tag">mcr-1</a> <a href="/bactopia-tools/tags/colistin-resistance" class="badge badge--primary badge--tag">colistin-resistance</a> <a href="/bactopia-tools/tags/antimicrobial-resistance" class="badge badge--primary badge--tag">antimicrobial-resistance</a> <a href="/bactopia-tools/tags/fasta" class="badge badge--primary badge--tag">fasta</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

Sequence variation analysis of mcr-1 genes (mobilized colistin resistance).

This Bactopia Tool uses [mcroni](https://github.com/liampshaw/mcroni) to identify _mcr-1_ genes in
assemblies and report sequence variations. If _mcr-1_ is found, the variations will be reported
and available in an output FASTA file.

## Usage

Bactopia CLI:

```bash
bactopia --wf mcroni \
  --bactopia /path/to/your/bactopia/results
```

Nextflow:

```bash
nextflow run bactopia/bactopia/workflows/bactopia-tools/mcroni/main.nf \
  --bactopia /path/to/your/bactopia/results
```
