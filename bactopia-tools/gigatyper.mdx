# gigatyper

**Tags:** <a href="/bactopia-tools/tags/mlst" class="badge badge--primary badge--tag">mlst</a> <a href="/bactopia-tools/tags/typing" class="badge badge--primary badge--tag">typing</a> <a href="/bactopia-tools/tags/multi-scheme" class="badge badge--primary badge--tag">multi-scheme</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

Run all available MLST schemes for a species against an assembly

This Bactopia Tool uses [GigaTyper](https://github.com/rpetit3/gigatyper) to run all available mlst schemes for a species against an assembly.

## Usage

```bash
bactopia --wf gigatyper \
  --bactopia /path/to/your/bactopia/results
```
