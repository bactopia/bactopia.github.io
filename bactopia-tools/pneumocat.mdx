# pneumocat

**Tags:** <a href="/bactopia-tools/tags/streptococcus-pneumoniae" class="badge badge--primary badge--tag">streptococcus-pneumoniae</a> <a href="/bactopia-tools/tags/capsular-typing" class="badge badge--primary badge--tag">capsular-typing</a> <a href="/bactopia-tools/tags/pneumocat" class="badge badge--primary badge--tag">pneumocat</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

Capsular type assignment to Streptococcus pneumoniae from sequence reads.

This Bactopia Tool uses [PneumoCaT](https://github.com/ukhsa-collaboration/PneumoCaT) to assign capsular
type to _Streptococcus pneumoniae_ from sequence reads for epidemiological typing.

## Usage

```bash
bactopia --wf pneumocat \
  --bactopia /path/to/your/bactopia/results
```
