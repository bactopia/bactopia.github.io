# pasty

**Tags:** <a href="/bactopia-tools/tags/fasta" class="badge badge--primary badge--tag">fasta</a> <a href="/bactopia-tools/tags/serogrouping" class="badge badge--primary badge--tag">serogrouping</a> <a href="/bactopia-tools/tags/pseudomonas-aeruginosa" class="badge badge--primary badge--tag">pseudomonas-aeruginosa</a> <a href="/bactopia-tools/tags/typing" class="badge badge--primary badge--tag">typing</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

In silico serogrouping of Pseudomonas aeruginosa isolates.

This Bactopia Tool uses [pasty](https://github.com/rpetit3/pasty) for
serogrouping of _Pseudomonas aeruginosa_ isolates from genome assemblies.

## Usage

Bactopia CLI:

```bash
bactopia --wf pasty \
  --bactopia /path/to/your/bactopia/results
```

Nextflow:

```bash
nextflow run bactopia/bactopia/workflows/bactopia-tools/pasty/main.nf \
  --bactopia /path/to/your/bactopia/results
```
