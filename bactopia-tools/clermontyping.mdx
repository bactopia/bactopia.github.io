# clermontyping

**Tags:** <a href="/bactopia-tools/tags/e-coli" class="badge badge--primary badge--tag">e-coli</a> <a href="/bactopia-tools/tags/phylotyping" class="badge badge--primary badge--tag">phylotyping</a> <a href="/bactopia-tools/tags/phylogroups" class="badge badge--primary badge--tag">phylogroups</a> <a href="/bactopia-tools/tags/clermontyping" class="badge badge--primary badge--tag">clermontyping</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

In silico phylotyping of Escherichia genus.

This Bactopia Tool uses [ClermonTyping](https://github.com/happykhan/ClermonTyping)
to conduct _in silico_ prediction of phylotype for _Escherichia_ genomes. It uses
genome assemblies to assign them to _E. albertii_, _E. fergusonii_, _Escherichia_
clades I–V, _E. coli sensu stricto_ as well as to the main _E. coli_ phylogroups.

## Usage

Bactopia CLI:

```bash
bactopia --wf clermontyping \
  --bactopia /path/to/your/bactopia/results
```

Nextflow:

```bash
nextflow run bactopia/bactopia/workflows/bactopia-tools/clermontyping/main.nf \
  --bactopia /path/to/your/bactopia/results
```
