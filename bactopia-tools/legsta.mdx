# legsta

**Tags:** <a href="/bactopia-tools/tags/legionella-pneumophila" class="badge badge--primary badge--tag">legionella-pneumophila</a> <a href="/bactopia-tools/tags/sbt" class="badge badge--primary badge--tag">sbt</a> <a href="/bactopia-tools/tags/typing" class="badge badge--primary badge--tag">typing</a> <a href="/bactopia-tools/tags/fasta" class="badge badge--primary badge--tag">fasta</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

Sequence Based Typing (SBT) of Legionella pneumophila.

This Bactopia Tool uses [legsta](https://github.com/tseemann/legsta) for
_in silico_ _Legionella pneumophila_ Sequence Based Typing (SBT).

## Usage

```bash
bactopia --wf legsta \
  --bactopia /path/to/your/bactopia/results
```
