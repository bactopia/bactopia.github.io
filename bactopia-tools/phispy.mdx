# phispy

**Tags:** <a href="/bactopia-tools/tags/prophage" class="badge badge--primary badge--tag">prophage</a> <a href="/bactopia-tools/tags/phage" class="badge badge--primary badge--tag">phage</a> <a href="/bactopia-tools/tags/annotation" class="badge badge--primary badge--tag">annotation</a> <a href="/bactopia-tools/tags/bacterial" class="badge badge--primary badge--tag">bacterial</a> <a href="/bactopia-tools/tags/archaeal" class="badge badge--primary badge--tag">archaeal</a> <a href="/bactopia-tools/tags/bactopia-tool" class="badge badge--primary badge--tag">bactopia-tool</a> 

Prediction of prophages in bacterial and archaeal genomes.

This Bactopia Tool uses [PhiSpy](https://github.com/linsalrob/PhiSpy) to identify prophages
from bacterial and archaeal genomes using machine learning approaches.

## Usage

Bactopia CLI:

```bash
bactopia --wf phispy \
  --bactopia /path/to/your/bactopia/results
```

Nextflow:

```bash
nextflow run bactopia/bactopia/workflows/bactopia-tools/phispy/main.nf \
  --bactopia /path/to/your/bactopia/results
```

### PhiSpy Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `--phispy_number` | integer | `5` | Number of consecutive genes in a region of window size that must be prophage genes to be called |
| `--phispy_mincontigsize` | integer | `5000` | Minimum contig size (in bp) to be included in the analysis. Smaller contigs will be dropped. |
| `--phispy_windowsize` | integer | `30` | Window size of consecutive genes to look through to find phages |
| `--phispy_nonprophage_genegaps` | integer | `10` | The number of non phage genes between prophages |
| `--phispy_phage_genes` | integer | `1` | The minimum number of genes that must be identified as belonging to a phage for the region to be included |
| `--phispy_randomforest_trees` | integer | `500` | Number of trees generated by Random Forest classifier |
| `--phispy_opts` | string |  | Extra options in quotes for PhiSpy |
